A = np.array([
    [1, 2],
    [3, 4]
])

B = np.array([
    [5, 6],
    [7, 8]
])



✅ How each element is computed (core idea)

Each element of C is:

C[i, j] = (row i of A) · (column j of B)



✅
Step-by-step

C[0,0]
[1, 2] · [5, 7] = 1×5 + 2×7 = 19

C[0,1]
[1, 2] · [6, 8] = 1×6 + 2×8 = 22

C[1,0]
[3, 4] · [5, 7] = 3×5 + 4×7 = 43

C[1,1]
[3, 4] · [6, 8] = 3×6 + 4×8 = 50
