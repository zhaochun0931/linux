



# 1. Initialize your Go module (if you haven't already)
go mod init my-raft-db

# 2. Download the core Raft library
go get github.com/hashicorp/raft

# 3. Download the BoltDB storage driver
go get github.com/hashicorp/raft-boltdb/v2

# 4. Clean up and lock the dependencies
go mod tidy







Wipe everything clean: rm -rf data/

Start Node 1:
go run main.go -id node1 -raft 7000 -http 8000 -bootstrap

Start Node 2:
go run main.go -id node2 -raft 7001 -http 8001 -join localhost:8000

Start Node 3:
go run main.go -id node3 -raft 7002 -http 8002 -join localhost:8000









Step 2: Insert New Data
Open a second terminal and use the /set API to insert data into your local DB:


curl "http://localhost:8000/set?key=username&value=admin"
curl "http://localhost:8000/set?key=theme&value=dark_mode"


Step 3: Verify Data


curl "http://localhost:8000/get?key=username"




curl "http://localhost:8000/get?key=theme"



curl -s http://localhost:8000/info



watch -n 1 'curl -s http://localhost:8000/info;curl -s http://localhost:8001/info;curl -s http://localhost:8002/info'





Safely remove Node 3:
curl "http://localhost:8000/remove?id=node3"

Safely remove Node 2:
curl "http://localhost:8000/remove?id=node2"

Kill Node 2 and Node 3 (Ctrl+C in their terminals).

Check Node 1: Notice it is still running, still the Leader, and completely unfazed because it safely adjusted its quorum!
