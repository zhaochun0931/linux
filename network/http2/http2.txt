| Aspect           | HTTP/1.1                                           | HTTP/2                                        |
| ---------------- | -------------------------------------------------- | --------------------------------------------- |
| Protocol         | Text                                               | Binary                                        |
| Connection usage | 1 request per TCP connection (pipelining optional) | Multiplex multiple streams per TCP connection |
| Headers          | Sent plain each request                            | Compressed (HPACK)                            |
