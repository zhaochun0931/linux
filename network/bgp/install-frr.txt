After a fresh install, starting FRR will do nothing. This is because daemons must be explicitly enabled by editing a file in your configuration directory.













# how to enable the bgp daemon

open the file /etc/frr/daemons and change the bgpd to yes

bgpd=yes





systemctl daemon-reload

systemctl restart frr







# how to configure the bgp 

vtysh


configure terminal

router bgp [your_AS_number]
neighbor [neighbor_IP] remote-as [neighbor_AS_number]



show running-config
