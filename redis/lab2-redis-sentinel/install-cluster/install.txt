helm repo add bitnami https://charts.bitnami.com/bitnami
helm repo update

# Install with Sentinel enabled
helm install redis-ha bitnami/redis \
  --set sentinel.enabled=true \
  --set replica.replicaCount=3




2. Verify the Pod Structure
Once deployed, check your pods. You will see a StatefulSet (e.g., redis-ha-node).

Each pod contains:

Redis container: Handles the data (Port 6379).

Sentinel container: Monitors the nodes and handles quorum (Port 26379).





kubectl get statefulsets



kubectl get pods


kubectl get svc
