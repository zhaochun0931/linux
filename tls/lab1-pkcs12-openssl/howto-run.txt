mkdir certs


cd certs



Part 1: Generate a Keystore


    Step 1: Generate a private key and a self-signed certificate

    This command creates a 4096-bit RSA private key and a certificate valid for 365 days.



    openssl req -x509 -newkey rsa:4096 -keyout private_key.pem -out certificate.pem -days 365 -nodes








    Step 2: Combine the key and certificate into a PKCS12 Keystore

    This command packages the PEM files into a .p12 file that Java can read.


    openssl pkcs12 -export -out keystore.p12 -inkey private_key.pem -in certificate.pem -name "myalias"





Part 2: Generate a Truststore


    keytool -importcert -alias "trustedcert" -file certificate.pem -keystore truststore.p12 -storetype PKCS12



keytool -list -keystore truststore.p12 -storetype PKCS12
