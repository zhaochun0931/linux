Step 1: Generate the Private Key and Self-Signed Certificate
We start by creating a standard RSA key and a certificate valid for 365 days.


openssl req -newkey rsa:2048 -nodes -keyout server.key -x509 -days 365 -out server.crt \
    -subj "/C=US/ST=State/L=City/O=Organization/OU=IT/CN=172.16.204.139"




Step 2: Combine into a PKCS12 Bundle
Java's keytool cannot easily import a raw private key and certificate directly. We must first "bundle" them into the PKCS12 format (.p12), which is a portable standard.

openssl pkcs12 -export -in server.crt -inkey server.key \
               -out server.p12 -name "gemfire-key" \
               -passout pass:password



Step 3: Convert PKCS12 to JKS
Now we use the Java keytool utility (included with your JDK) to convert that .p12 file into a .jks file that GemFire understands.

keytool -importkeystore -srckeystore server.p12 -srcstoretype PKCS12 \
        -destkeystore gemfire.jks -deststoretype JKS \
        -srcstorepass password -deststorepass password
